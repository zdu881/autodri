# Driver Monitoring + Gaze ONNX

This repository contains two modules:

- `driver_monitor`: driver monitoring utilities (gaze tracking, hand-on-wheel, checks)
- `gaze_onnx`: ONNX-based gaze estimation pipeline

Generated artifacts (videos, csv outputs, debug images, snapshots) are kept under each module’s `output/` or `snaps*` folders and are ignored by git.
